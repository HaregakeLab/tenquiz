# Vibe Scaffold Laravel

**AI と対話するだけで Web アプリが作れるテンプレート**

プログラミングの知識は一切不要です。AI ツール（Claude Code、Cursor など）に「こんなアプリを作りたい」と日本語で伝えるだけで、Web アプリケーションが作れます。

## はじめかた

1. **「Use this template」** を押して、組織のアカウントに新しいリポジトリを作る
2. **AI ツールをインストールする** - 下の「対応 AI ツール」からお好きなものを選ぶ
3. **プロジェクトを AI ツールで開く**
4. **AI に「セットアップして」と言う** - あとは AI が全部やってくれます
5. 作りたいものを AI に伝える
6. **AI に「公開して」と言う** - 自動でサーバーに公開される

**例：**
- 「お問い合わせフォームを作って」
- 「商品の一覧ページを作って」
- 「デザインを青系の色に変えて」

## 対応 AI ツール

| ツール | 特徴 |
|--------|------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview) | ターミナルで動く AI アシスタント。指示を出すだけで全部やってくれる |
| [Cowork](https://claude.ai/download) | Claude Desktop アプリからコードを編集できる |
| [Cursor](https://cursor.com/) | AI が組み込まれたエディタ。チャットで指示を出せる |
| [Codex](https://openai.com/index/introducing-codex/) | OpenAI のクラウド型 AI 開発ツール |
| [Antigravity](https://antigravity.codes/) | Google Gemini ベースの AI 開発ツール |
| [GitHub Copilot](https://github.com/features/copilot) | VS Code と組み合わせて使う AI アシスタント |

どれを選んでも大丈夫です。
