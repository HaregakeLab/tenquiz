# Vibe Scaffold Laravel

**AI と対話するだけで Web アプリが作れるテンプレート**

プログラミングの知識は一切不要です。AI ツール（Claude Code、Cursor など）に「こんなアプリを作りたい」と日本語で伝えるだけで、Web アプリケーションが作れます。

## このドキュメントについて

このサイトでは、テンプレートの使い方を説明しています。

- **[はじめかたガイド](getting-started.md)** — 最初のセットアップ手順
- **[AI で開発する方法](development.md)** — AI への伝え方のコツ
- **[データの保存について](database.md)** — データを保存したいとき
- **[公開する方法](deployment.md)** — 作ったアプリを公開する
- **[困ったときは](faq.md)** — よくある質問

## クイックスタート

1. テンプレートから **「Use this template」** で新しいリポジトリを作る
2. AI ツールで開いて **「セットアップして」** と言う
3. 作りたいものを AI に伝える
4. **「公開して」** と言えばサーバーに自動デプロイ
